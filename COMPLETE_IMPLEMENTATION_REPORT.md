### Using Consolidated Chunking
```python
from src.rag.chunking import create_chunker

# Simple usage
chunker = create_chunker("recursive", chunk_size=512)
chunks = chunker.chunk({
    "id": "doc1",
    "content": "Your document text here..."
})

# All strategies available:
# - "fixed": Fixed-size chunking
# - "recursive": Recursive character chunking (recommended)
# - "semantic": Embedding-based semantic chunking
# - "hierarchical": Parent-child hierarchical chunking
# - "code": Language-aware code chunking
# - "token_aware": Token-precise tiktoken chunking
```
