# Hands-on LLM Mastery: The Complete Series

Welcome to the **AI-Mastery-2026** official LLM tutorial series. This guide provides a deep-dive, code-first approach to understanding, building, and deploying Large Language Models and their ecosystem.

## 📖 Series Roadmap

### [Chapter 1: The Heart of LLMs - Attention & Transformers](./01_attention_and_transformers.md)
*   **Concepts**: Scaled Dot-Product, Multi-Head Attention, RoPE.
*   **Architectures**: Implementation of BERT and GPT-2 from scratch.
*   **Hands-on**: Building your first encoder-decoder block.

### [Chapter 2: Scaling Knowledge - Enterprise RAG](./02_enterprise_rag.md)
*   **Concepts**: Semantic Chunking, Dense vs Sparse Retrieval, Hybrid Fusion.
*   **Advanced**: Query Rewriting, Reranking, and Knowledge Graphs.
*   **Hands-on**: Building a production-ready RAG pipeline for the project docs.

### [Chapter 3: The Reasoning Engine - AI Agents](./03_ai_agents.md)
*   **Concepts**: Reactive vs Deliberative agents, the ReAct pattern.
*   **Orchestration**: Multi-agent systems and message routing.
*   **Hands-on**: Coding a research agent that uses tools.

### [Chapter 4: Shipping to Production - Guardrails & Quality](./04_production_guardrails.md)
*   **Concepts**: Hallucination prevention, source citation, CX scoring.
*   **Safety**: Content scope enforcement and automated human handoff.
*   **Hands-on**: Implementing an automated "LLM-as-Judge" evaluation suite.

## 🛠️ Environment Setup
Ensure you have the virtual environment activated:
```bash
# Windows
.venv\Scripts\activate
# Unix
source .venv/bin/activate
```

Install dependencies if you haven't already:
```bash
make install
```
