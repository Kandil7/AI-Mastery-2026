Choose your learning journey:

### Path 1: AI Foundations (4-6 weeks)

```
Week 1-2: Mathematics
├── Linear Algebra
├── Calculus
└── Probability

Week 3-4: Programming
├── Python for AI
├── NumPy Essentials
└── Data Visualization

Week 5-6: Machine Learning
├── Linear Regression
├── Logistic Regression
└── Decision Trees
```

**Start:** [Getting Started](guides/getting-started.md)

---

### Path 2: ML Practitioner (12-16 weeks)

```
Week 1-4: Foundations (see Path 1)

Week 5-8: Classical ML
├── SVM
├── Clustering
├── Ensemble Methods
└── PCA

Week 9-12: Deep Learning
├── Neural Networks
├── Backpropagation
├── CNN
└── RNN/LSTM

Week 13-16: Projects
├── Build a classifier
├── Image recognition
└── Time series prediction
```

**Start:** [Classical ML Tutorials](tutorials/intermediate/)

---

### Path 3: LLM Engineer (20-24 weeks)

```
Week 1-8: Foundations (see Path 2)

Week 9-12: Transformers
├── Attention Mechanisms
├── Transformer Architecture
├── Tokenization
└── Implementation

Week 13-16: LLM Engineering
├── Fine-Tuning
├── LoRA/QLoRA
├── Prompt Engineering
└── Evaluation

Week 17-20: RAG Systems
├── RAG Fundamentals
├── Vector Databases
├── Advanced RAG
└── RAG Evaluation

Week 21-24: Production
├── Deployment
├── Monitoring
├── Optimization
└── Scale
```

**Start:** [LLM Engineering Tutorials](tutorials/intermediate/finetuning-llm.md)

---

### Path 4: AI Architect (24-30 weeks)

```
Week 1-20: LLM Engineer Path

Week 21-24: AI Agents
├── Agent Architecture
├── Tool Use

└── LangGraph

Week 25-28: Advanced Topics
├── Multi-Modal Systems
├── Distributed Training
├── Model Optimization
└── Security

Week 29-30: Capstone Project
└── Build production system
```

**Start:** [AI Agents Tutorial](tutorials/intermediate/ai-agents.md)
