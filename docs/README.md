## рџљЂ Getting Started

### Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Kandil7/AI-Mastery-2026.git
   cd AI-Mastery-2026
   ```

2. **Install dependencies:**
   ```bash
   make install
   ```

3. **Verify installation:**
   ```bash
   make test
   ```
