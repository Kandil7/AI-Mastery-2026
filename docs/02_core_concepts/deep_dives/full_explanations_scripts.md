# Scripts (scripts/*.py)

## scripts/case_study_payloads.py

### _print(title: str, payload: dict) -> None
- Prints a title and JSON payload to stdout.
- Uses `ensure_ascii=True` to keep ASCII output.

### main() -> None
- Prints example payloads for each case study:
  - Customer Support
  - Compliance & Legal
  - Engineering KB
  - Sales Enablement
  - Incident Response
  - HR Policy

### Script usage
- Run: `python research/week5-backend/week5_backend/scripts/case_study_payloads.py`
