# Full Project Explanations (Week5 Backend)

This folder contains detailed explanations for every function and class in the
`research/week5-backend/week5_backend` package.

Files
- `agents.md` -> agentic planning, tools, verification
- `app.md` -> FastAPI app, routers, and request/response schemas
- `core.md` -> settings, factories, logging, telemetry
- `evaluation.md` -> evaluation datasets and harness
- `pipelines.md` -> online query and offline indexing pipelines
- `providers.md` -> LLM and embeddings providers
- `rag.md` -> chunking, retrieval, reranking, answering, citations
- `storage.md` -> vector store backends
- `scripts.md` -> runnable scripts
- `run_api.md` -> dev server entry point
