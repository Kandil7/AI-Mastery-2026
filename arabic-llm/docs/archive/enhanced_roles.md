## 9. Usage Example

```python
from schema_enhanced import (
    TrainingExample, Role, Skill, Level, Domain, 
    Style, TaskType, validate_example, ROLE_SKILL_MAP
)

# Create a faqih example
example = TrainingExample(
    instruction="ما حكم صلاة المسافر في الفقه الشافعي؟",
    input="",
    output="صلاة المسافر في الفقه الشافعي: يقصر الرباعية...",
    role=Role.FAQIH,
    skills=[Skill.FIQH],
    level=Level.ADVANCED,
    domain=Domain.ISLAMIC_STUDIES,
    style=Style.SCHOLARLY,
    task_type=TaskType.FATWA,
    difficulty=3,
    book_category="الفقه الشافعي",
    author_name="الإمام الشافعي",
)

# Validate
errors = validate_example(example)
if not errors:
    print("✓ Example is valid!")
```
