## 📚 Learning Paths

### Tier 1: Beginner Foundation (Weeks 1-4)

**Goal:** Build strong fundamentals in math, Python, and ML basics

**Prerequisites:** Basic programming knowledge (any language)

**Time Commitment:** 80-120 hours (20-30 hours/week)

**Courses:**
1. **Mathematics for AI** - Linear algebra, calculus, probability
2. **Python for ML** - NumPy, Pandas, Matplotlib, scikit-learn
3. **Introduction to Neural Networks** - Perceptrons, backpropagation
4. **NLP Fundamentals** - Text processing, embeddings

**Outcome:** Ready to tackle transformer architectures and LLM fundamentals

[**Start Tier 1 →**](./learning_paths/tier1_fundamentals/README.md)

---

### Tier 2: LLM Scientist (Weeks 5-8)

**Goal:** Understand LLM architecture, training, and fine-tuning at a deep level

**Prerequisites:** Tier 1 completion or equivalent knowledge

**Time Commitment:** 160-200 hours (40-50 hours/week)

**Courses:**
1. **Transformer Architecture** - Attention, encoders, decoders
2. **LLM Pretraining** - Data preparation, distributed training
3. **Supervised Fine-Tuning (SFT)** - Instruction tuning, adapters
4. **Preference Alignment** - RLHF, DPO, reward modeling
5. **Evaluation Methods** - Benchmarks, metrics, testing

**Outcome:** Can train and fine-tune LLMs from scratch

[**Start Tier 2 →**](./learning_paths/tier2_llm_scientist/README.md)

---

### Tier 3: LLM Engineer (Weeks 9-12)

**Goal:** Build production-ready LLM applications (RAG, agents, systems)

**Prerequisites:** Tier 2 completion or strong LLM fundamentals

**Time Commitment:** 200-280 hours (50-70 hours/week)

**Courses:**
1. **Running LLMs** - Inference, quantization, optimization
2. **Vector Storage** - Embeddings, databases, indexing
3. **RAG Systems** - Retrieval, generation, orchestration
4. **Advanced RAG** - Multi-hop, hybrid, temporal, graph-enhanced
5. **AI Agents** - Tool use, planning, multi-agent systems
6. **LLM Security** ⭐ - Prompt injection, defenses, safety

**Outcome:** Can build sophisticated LLM applications

[**Start Tier 3 →**](./learning_paths/tier3_llm_engineer/README.md)

---

### Tier 4: Production & DevOps (Weeks 13-17)

**Goal:** Deploy, monitor, and maintain LLM systems in production

**Prerequisites:** Tier 3 completion + capstone project started

**Time Commitment:** 40-80 hours (10-20 hours/week)

**Courses:**
1. **Deployment Patterns** - Docker, Kubernetes, scaling
2. **Inference Optimization** - Caching, batching, streaming
3. **Monitoring & Observability** ⭐ - Metrics, tracing, alerting
4. **Cost Optimization** ⭐ - Token budgeting, model cascading
5. **MLOps for LLMs** - CI/CD, versioning, continuous evaluation
6. **Capstone Project** - End-to-end production system

**Outcome:** Production-ready LLM engineer

[**Start Tier 4 →**](./learning_paths/tier4_production/README.md)
